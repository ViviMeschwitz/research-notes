## 哲学的基盤

### クレマン・ロセ『現実とその分身』
- 人間は受け入れ難い現実に直面すると「分身（ダブル）」を作り出し逃避する
- 主人公の分身構造はロセの「心理学的錯覚」の実践
- 主人公は**自覚的に**錯覚に入っていく——これが通常の解離と異なる点

### デカルト「我思う、ゆえに我あり」との接続
- デカルトは懐疑によって錯覚を排除し確実な自己の基盤を探した
- 主人公は逆方向——錯覚の中にあえて入っていく
- **どの「我」が考えているのか**という問いが核心

### スピノザ（接続予定）
- 感情＝外部の原因によって動かされる状態＝システムへの支配
- 自由な人間＝感情を外から眺め秩序の中に位置づけられる人間
- 主人公の理想像＝感情をも集合として定義できる人間

## 中心的対立構造

```
理想像（天才）              生活者（凡人）
　↓                            ↓
概念に色と平面を持つ         感情にエントロピーが高い
内的宇宙に秩序がある         システムに支配される
集合を定義できる             境界が溶けている
主体である                   客体である
```

### 葛藤の性質
- 両極の間を**振れ続ける**こと自体が苦しみの本質
- 天才と確信できれば進める、凡人と諦めれば休める
- しかし振れ幅の中にいる限りどちらにも着地できない
- 容姿という**毎日見える鏡**がこの振れ幅を日常的に刺激する

## 主人公の内的宇宙（概念モデル）

主人公の頭の中では概念が以下の構造を持つ：

- 1つの命題 → 3次元空間上の1つの無限平面を生成
- 集合 → 境界のある有限平面（命題と同型：真偽＝属するか否か）
- 集合演算 → 異なる平面を重ねる＝色を混ぜる
- 概念 → 色として存在する

**核心：秩序の人間が、秩序のない感情に飲み込まれる恐怖**

→ [[i_entropy_emotion]]

## 重要な場面：電車の窓

> 電車が地下から地上に出た瞬間、窓が鏡になった。
>
> そこに顔があった。自分の顔だ。わかっている。しかし一瞬、他人の顔に見えた。
>
> 目が合う。
>
> 逸らせない。逸らすことが負けな気がして、でも見続けることも怖くて、結局僕は窓枠の外の暗闘を見るふりをしながら、その顔を見ていた。
>
> 悪くはない、と思う。思おうとする。しかし「悪くない」と「美しい」の間には、宇宙ほどの距離がある。僕が欲しいのは後者だ。いや、正確には——僕が欲しいのは、その顔が**あの人間**に相応しいかどうか、という問いへの答えだ。
>
> 答えは出ない。
>
> 電車が駅に着く。扉が開く。風が入る。窓の顔が消える。

---

> 改札を出ると、街灯の間隔が広くなった。足音だけが聞こえる。
>
> 不安はどこまでが不安かわからない。絶望はもっとわからない。僕の足音が、暗闘の中に少しずつ吸い込まれていく気がした。

## ニュートン的探求路線の交差点

*彼のさまざまな探求路線が出会うところ、彼がそれらの不一致をもっと根本的な一体性へと統合しようと試みるとき、まさにそのとき彼は最も偉大な洞察を獲得したのだ。*
— The Janus Faces of Genius

現時点での路線：
```
数学（集合・位相）→ 概念の幾何学的直観 → 主人公の内的宇宙
情報科学（Shannon entropy）→ 感情の境界喪失 → 天才vs凡人の対立
哲学（デカルト・スピノザ）→ どの「我」が考えるか → 感情の支配vs自由
物語 → ［未決：錯覚に潜るか、秩序を取り戻すか］
```

これらの不一致の統合が、物語の結末を決定する。
