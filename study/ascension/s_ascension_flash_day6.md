## 3. 数学的基盤

### HTMLとは何か

深層の概念を、世界が読める形に符号化する手段。送信分布 p(x) の一形態。

### 3次元空間の2次元投影

- x軸：横
- y軸：縦
- z軸：奥行き（遠近）

2Dの画面に3次元空間を投影——次元を1つ落とす。

### 透視投影の基本式

$$size = \frac{k}{z}$$

- zが大きい（遠い）→ sizeが小さい
- zが小さい（近い）→ sizeが大きく、鮮明

### 移動の時間関数

$$z(t) = z_{max} - t \cdot v$$

- $z_{max}$：最初の距離（遠い）
- $v$：速度
- $t$：時間

## 4. 一つのサイクル

```
z_max → 0 → 消散 → 次の正方形
```

1. 正方形が遠くに生成される（大きなz、小さいsize）
2. zが減少しながら手前に迫ってくる（sizeが増大）
3. z=0で消散（脳への染み込み）
4. 新しい色の正方形が生成される
5. 繰り返す

## 6. 速度設計：非生物的な動き

「物理的リアルさを絶妙に逸脱した非生物的な動き」

生物は近づくほど慎重になる。Ascensionはその逆——近づくほど制御を失っていく。

```
遠い（z大）→ ゆっくり・微振動
      ↓
中間  → 突然加速
      ↓
近い（z小）→ さらに加速・振動が激しくなる
      ↓
z=0  → 消散
```

**アンキャニーバレーの応用：** 生き物のように動くが、生き物ではないとわかる。それが不安と聖性を同時に生む。
