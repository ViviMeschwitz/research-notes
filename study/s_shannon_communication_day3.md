## 色と直観の記録

### BSC（二値対称通信路）：紫
- 「化ける・歪む・不確かさ」の色
- 遷移確率の非対称性・不安定さを表す

## Shannonの通信モデル

```
送信者 → [送信機] → チャネル → [受信機] → 受信者
                       ↑
                     ノイズ（紫）
```

### 確率変数の定義
- $X$：送信機が生成した**信号そのもの**（装置ではない）
- $Y$：受信機が受け取った**信号そのもの**
- $p(x)$：送信機が各信号を送る**確率分布**（個別の値ではなく分布全体）

---

## 0と1について

- 0と1は信号の最小単位（bit）
- 文章・画像・音声はすべて0と1の列に変換される（符号化）
- 例：文字「A」→ `01000001`
- 物理的信号は「ある/なし」「高/低」の二値が実装しやすい

---

## 二値対称通信路（BSC）

$$p(Y=1|X=0) = p(Y=0|X=1) = \varepsilon$$

「$0$ が化ける確率」＝「$1$ が化ける確率」——これがBSCの定義。

**重要な区別：**

| | 意味 | 誰が決めるか |
|---|---|---|
| BSC | 両方向の遷移確率が等しい | チャネルの性質（固定） |
| $p(x)=0.5$ | 0と1を均等に送る | 送信者が選ぶ |

```
X=0 →（1-ε）→ Y=0
X=0 →（ε）  → Y=1（化けた）
X=1 →（ε）  → Y=0（化けた）
X=1 →（1-ε）→ Y=1
```

---

## 相互情報量 $I(X;Y)$

$$I(X;Y) = H(X) - H(X|Y)$$

| 要素 | 意味 |
|------|------|
| $H(X)$ | 受け取る前の不確実性 |
| $H(X\|Y)$ | 受け取った後も残る不確実性 |
| $I(X;Y)$ | 解消された不確実性＝伝達に成功した情報量 |

```
H(X)［送信前の不確実性］
　↓ Y を受け取る
H(X|Y)［残った不確実性］
　↓
I(X;Y) = 解消された量
```

### 情報を伝えるとは
送り手に起きた変化（混沌→整理）を受け手にも体験させること。
＝ 受け手のエントロピーを削減すること。

---

## 通信路容量 $C$

$$C = \max_{p(x)} I(X;Y)$$

- $p(x)$ は**送信分布全体**の最適化
- $H(X|Y)$ はチャネルの性質で固定（制御不能）
- $H(X)$ は送信分布で制御可能
- $H(X)$ を最大にする分布＝均等分布

### なぜ均等分布が最適か
送る情報の確率に偏りがあると、受け手は「また $0$ が来るだろう」と予測できる。受け取る前からエントロピーが低い——情報を受け取っても削減できる不確実性が少ない。均等分布のとき $H(X)$ が最大（$=1$ bit）になる。

---

## パズルの比喩

```
送り手：パズルの部品（0と1）を送る
チャネル：部品が化けるかもしれない（紫・BSC）
受け手：届いた部品からパズルを組み立てる
```

送信分布＝パズルの部品をどのように送信するか（できるだけ正確に情報を伝えるため）。組み立てるのは受け手。

---

## 誤り訂正符号（error-correcting code）

ノイズがあっても正確に伝えるために、送り手は**冗長性（redundancy）**を加える。

### 繰り返し符号の例
$$0 \to 000, \quad 1 \to 111$$

受け手は多数決で判断：

| 届いた信号 | 多数決 | 判断 |
|-----------|--------|------|
| $000$ | $0$が3つ | $0$ |
| $001$ | $0$が2つ | $0$ |
| $011$ | $1$が2つ | $1$ |
| $111$ | $1$が3つ | $1$ |

1bitが化けても正しく復元できる。

**トレードオフ：** 3倍の信号を送る → 伝送速度が $\dfrac{1}{3}$ になる。

---

## Shannon の通信路符号化定理

> 伝送速度を $C$（通信路容量）以下に保てば、冗長性を増やさなくても誤り率を $0$ にできる賢い符号が存在する。

これがShannonの最大の発見——ノイズがあっても原理的に誤りをゼロにできる。

## 有限性を振り払う原則

微分積分の時に有限性の考えを振り払ったように、Shannonの理論でも有限性を振り払うことが重要。一見直観に反するように見えることでも受け入れることが数学的思考の核心だ。

```
微分：Δt→0（有限の差分を振り払う）
Shannon：n→∞（有限回の偏りを振り払う）
位相空間：ε→0（有限の距離を振り払う）
```

すべて**有限性の呪縛から解放される操作**だ。

→ Day 2の洞察「微分・積分は有限の概念とは本質的に異なる」と接続。
