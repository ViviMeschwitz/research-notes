## Field とは

Lullabyの中核機能。

> 書き手の思考と感覚の変化率を、色として可視化する場。

初期状態：真っ白。
書き手が概念・文章を入力するたびに、色が生まれ、動き、混ざる。

---

## 設計思想

### 数学的基盤（ε-δ論法との対応）

```
横軸（概念・時間・文脈）：δ が制御する範囲
縦軸（色・感情・身体感覚）：ε が制御する範囲
両者の連動 → 思考と感覚の微分
```

思考が「統合失調症的」に飛躍した瞬間——εが閾値を超えた瞬間——をCBDが検出し、境界を可視化する。

### 言語-概念-イメージモデルとの接続
- 概念は言語ではなくイメージ（色）として定着する
- チョムスキーの深層構造：概念をイメージで持ち、多言語をマッピングする
- Fieldは概念の多言語マッピングを視覚的に実現する

### 音楽との接続
- 音楽が「場の情報を書き換える」のと同じ操作をFieldが行う
- 書き手の思考がFieldに入力されると、その場の色が変わる
- 読み手はその色の変化を通じて概念を身体で感じる

---

## ユーザーの二つの役割

### 書き手として
- 自分の思考の色・変化率をリアルタイムで可視化しながら書く
- CBDが認知限界（entropy閾値）を検出し通知する
- 他のユーザーと思考の場を共有できる

### 読み手・学習者として
- 文章の視覚的イメージを掴む
- 概念が言語ではなくイメージとして定着する
- 書き手の思考プロセスを色の変化として追体験する

## 色の構造

| 状態 | 色 |
|------|------|
| 初期（空白） | 真っ白 |
| 集合（境界のみ） | 透明な境界 |
| metric導入 | 緑 |
| open ball | 黄 |
| 位相空間 | 青 |
| entropy増加（不安） | 境界溶解 |
| entropy閾値超過（絶望） | 紫 |
| 統合失調症的飛躍 | 茶・紫 |

---

## CBDとの統合

```
Field（色の変化）
　↓ 変化率がε閾値を超える
CBD検出
　↓
書き手への通知：「認知境界に近づいています」
　↓
思考の再編成を促す
```

## Truth is One との接続

数学（色・境界）・物理（変化率）・情報科学（entropy）・音楽（場の書き換え）・文学（身体感覚）——すべてがFieldという一つの機能に収束する。
