## 問い

言語は概念に対して完全な1対1対応ではない。
日本語で理解した概念を日本語ではなくイメージ（色・形）に落とし、
そのイメージを英語で引っ張るときに英単語を用いるべきではないか。

---

## 概念表象モデル

言語は概念へのアクセスキーに過ぎない。
概念そのものは言語から独立して存在する。
```
日本語「補集合」
英語「complement」　→　【概念】　←　イメージ（Aの外側の空間）
フランス語「complément」
```

3言語が同じ概念を指す。
概念をイメージで持てば3言語が自然に繋がる。

---

## アイデア

### 1. 概念に固有の色を割り当てる
- entropy　→　灰色のグラデーション（不確かさ・霧）
- set　→　青い輪郭（境界のある空間）
- complement　→　外側の空間（輪郭の外）

### 2. 形・空間で概念を持つ
集合は「境界のある空間」、補集合は「その外側」。
数式を見たとき言語を経由せず空間イメージが浮かぶ。

### 3. 多言語同時導入
新しい概念を学ぶとき、日本語・英語・フランス語を同時に見る。
どの言語も「第一言語」にならず、概念が中心に来る。

## 文法の学習方針

文法も「ルールの暗記」ではなく構造のイメージ化を原則にする。
```
文法ルール → 構造のイメージ → 概念との対応
```

### 英語の語順を構造として見る
```
【誰が】→【する】→【何を】→【どのように】
```

力の流れとして空間的にイメージする。
日本語との違いが「ルール」ではなく「構造の違い」として見える。

### チョムスキーの生成文法との対応

深層構造 = 概念・イメージ
表層構造 = 各言語の表現

深層構造を直接掴んで、各言語の表層構造をそこからマッピングする。

### 実践教材

伊藤和夫「英文解釈教室」
文を構造の塊として見る訓練。論文読解の基盤。
