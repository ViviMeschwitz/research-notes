## 問い

言語は概念に対して完全な1対1対応ではない。
日本語で理解した概念を日本語ではなくイメージ（色・形）に落とし、
そのイメージを英語で引っ張るときに英単語を用いるべきではないか。

---

## 概念表象モデル

言語は概念へのアクセスキーに過ぎない。
概念そのものは言語から独立して存在する。
```
日本語「補集合」
英語「complement」　→　【概念】　←　イメージ（Aの外側の空間）
フランス語「complément」
```

3言語が同じ概念を指す。
概念をイメージで持てば3言語が自然に繋がる。

---

## アイデア

### 1. 概念に固有の色を割り当てる
- entropy　→　灰色のグラデーション（不確かさ・霧）
- set　→　青い輪郭（境界のある空間）
- complement　→　外側の空間（輪郭の外）

### 2. 形・空間で概念を持つ
集合は「境界のある空間」、補集合は「その外側」。
数式を見たとき言語を経由せず空間イメージが浮かぶ。

### 3. 多言語同時導入
新しい概念を学ぶとき、日本語・英語・フランス語を同時に見る。
どの言語も「第一言語」にならず、概念が中心に来る。

→ 色付けパターンが豊か・多様
