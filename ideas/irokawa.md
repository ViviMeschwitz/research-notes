# 色河（いろかわ）設計ノート
*2026年3月14日 文豪アイデアセッション*

---

## 色河とは

文豪の中核にある「思考の地形図」。自分がこれまで読み・書き・考えてきたすべての文章が、一つの宇宙として可視化される場所。

---

## 2つのモード

### 俯瞰モード
空から地図を見るように、思考の全体像を見る。

- すべての文章が**輪郭のみの点**として表示される
- 色を選ぶと、その色が付与された点と川だけが染まる
- 複数色を選ぶと、そのパターンを持つ点だけが染まる
- 色の**順序**が重要（黄色→青と青→黄色は別物）

### 擬人化モード
気になる点をクリックすると、自分がその文章の中に入る。

- その文章の6ステップの記録、色、再現率、言い換えが見える
- そこから「チャネルを開通する」ことができる
- 俯瞰モードと同じ色操作ができる

---

## 点と川

### 点（文章）

```
選択前：  ○  輪郭のみ、透明
選択後：  ●  選択した色に染まる
```

### 川（チャネル）

```
川の色 = どの色から開通したチャネルかを示す

A ——黄色の川——→ B ——黄色の川——→ C
│
青の川
│
E ——青の川——→ F
```

---

## Version Systemとの関係

文章は縦（推敲）と横（チャネル）の両方向に広がる。

```
A（原文）
│  ├── チャネル1（黄色から）：B → C → D
│  └── チャネル2（青から）：E → F
↓ 推敲
A'（書き直し）
│  ├── チャネル3（黄色から）：G → H
│  └── チャネル4（赤から）：I → J
↓ 推敲
A''
   └── ...
```

**AからのチャネルとA'からのチャネルは別物。**
山岳地帯の川と大河川の川が別物であるように。

---

## チャネルとは

一つの文章を起点に、思考が連鎖していく流れ。

- 文章Aを読んで思ったことをBに書く
- BをさらにCに深める
- CからDが生まれる
- この連鎖がチャネル

チャネルは**色ごとに開通する**。同じ文章Aでも、黄色から開通したチャネルと青から開通したチャネルは全く別の川になる。

一つの長い文章では要素によって色が異なるため、色の数だけチャネルの入り口が生まれる。

---

## 色とは

単なるタグではなく、**思考のパターンを表す記号**。

- 順序が重要（黄色→青→赤は一つのパターン）
- 複数色を持つ文章は情報量が多い
- 同じ色パターンを持つ文章を横断して参照できる
- 過去に同じパターンで考えた時の思考過程が、現在の道標になる

---

## 色河が実現すること

```
現在の思考で方向性が定まらない時
        ↓
色河で同じ色パターンの過去の文章を探す
        ↓
その時の思考過程を参照する
        ↓
その分岐点に戻って別のルートを試す
        ↓
現在の思考の方向が定まる
```

過去の膨大な記録が、現在の思考実験の環境になる。

---

## 文豪の全体構造における位置づけ

```
文豪
├── 6ステップ        思考を身体に染み込ませる
├── Version System   縦：推敲の歴史を残す
├── チャネル         横：思考の連鎖を記録する
├── Books            本にまとめる
└── 色河             すべてを俯瞰する思考の地形図
```

6ステップ・Version System・チャネル・Booksは、すべて色河という地形図を作るための部品である。

---

*色河という名前の由来：色が流れる川。思考の色が時間とともに流れ、地形を作っていく。*
