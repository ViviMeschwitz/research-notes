## 毎日のスケジュール（6:2ルール）

**原則:** 6スロットはインプット（学習）、2スロットはアウトプット（出荷）。

## モード使い分けルール

```
コーディング（文豪の開発・バグ修正）→ Claude Code
学習対話（数学・哲学・情報理論の議論）→ 通常チャット
ツール操作（カレンダー・メール・ファイル作成）→ Cowork
精読・演習・ノート整理 → 自力（Claude不要）
```

**トークン節約の鉄則:**
- Coworkは最もトークンを消費する。ツール操作以外では使うな。
- 通常チャットで十分な対話にCoworkを使うな。
- 自力でできることにClaudeを使うな。
