# Research Notes

> 数学・物理・情報理論を統合した知識ネットワーク型研究ノートシステム

## 目標
人間の知能の本質を理解し、数学・物理・情報科学・AIを統合した新しい理論を構築する。

## 期間
2026年〜2036年（10年間・1日16時間）

## ノート種別

| プレフィックス | 種別 | 内容 |
|-------------|------|------|
| `s_` | Study | 学習内容・定理・証明 |
| `q_` | Question | 疑問・未解決問題 |
| `i_` | Idea | 研究アイデア・ひらめき |
| `r_` | Research | 仮説・数式・実験・研究 |
| `d_` | Daily | 日次研究ログ |

## フォルダ構造

```
research-notes/
├── daily/          # 日次研究ログ
│   └── YYYY/
│       └── YYYY-MM-DD.md
├── study/          # 学習ノート
├── questions/      # 疑問ノート
├── ideas/          # アイデアノート
├── research/       # 研究ノート
├── index/          # 索引
├── references/     # 参考文献
└── meta/           # システム設定・ルール
```

## リンク記法
ノート間リンク: `[[note_name]]`

## 数式記法
LaTeX: `$inline$` / `$$block$$`

## 知識ネットワーク構造
```
Study → Question → Idea → Research
```
